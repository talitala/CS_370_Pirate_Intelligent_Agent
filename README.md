# CS_370_Pirate_Intelligent_Agent
This project demonstrates a pirate intelligent agent using reinforcement learning in a Jupyter Notebook. The environment and training framework were provided, and I completed key parts of the Deep Q-Learning algorithm, focusing on Q-value updates and exploration logic so the agent could improve its decisions over time.

Brief Explanation of the Work

In this project, most of the pirate intelligent agent code was provided, including the environment setup, neural network structure, and overall training framework. My role was to complete specific sections of the reinforcement learning implementation. I focused on filling in the missing parts of the Deep Q-Learning algorithm, such as updating Q-values, implementing the epsilon-greedy strategy, and ensuring the training loop worked correctly.

Although I did not build the entire project from scratch, I worked directly with the core learning logic and made sure the agent could properly learn from rewards and improve over time. This helped me better understand how reinforcement learning models function in practice.
